# PassVault Pro – Detailed Explanation of All Imports

PassVault Pro is built entirely using Python's standard library.  
No external packages are required.  
Below is a fully detailed breakdown of every import used in the project, along with why it is needed and where it is used.

────────────────────────────────────────────────────────────────────────

IMPORT: sqlite3
PURPOSE:
    Provides an embedded SQL database engine stored in a single .db file.
    Requires no installation, no server, and is perfect for desktop apps.

WHAT IT DOES IN THIS PROJECT:
    • Creates the database file (passwords_master.db)
    • Creates the tables: passwords, settings
    • Inserts, updates, deletes, and fetches password entries
    • Stores hashed master password, salt, and iteration count
    • Supplies data to the Tkinter table (Treeview)

DEFINITION:
    “sqlite3 allows Python to store data permanently inside a small database file.”

────────────────────────────────────────────────────────────────────────

IMPORT: os
PURPOSE:
    Interacts with the operating system (file existence checks, paths).

WHAT IT DOES IN THIS PROJECT:
    • Checks if the database exists on first launch
    • Determines whether to create seed data

DEFINITION:
    “Used to check if the database file already exists.”

────────────────────────────────────────────────────────────────────────

IMPORT: random
PURPOSE:
    General-purpose randomness (NOT security grade).

WHAT IT DOES IN THIS PROJECT:
    • Creates random character combinations during password generation
      (final validation of characters is enforced manually)

DEFINITION:
    “Used for generating random passwords.”

────────────────────────────────────────────────────────────────────────

IMPORT: string
PURPOSE:
    Provides convenient collections of characters:
        - ascii_lowercase
        - ascii_uppercase
        - digits
        - punctuation symbols

WHAT IT DOES IN THIS PROJECT:
    • Supplies character sets used by the password generator
    • Makes password generation more flexible and neat

DEFINITION:
    “Gives groups of characters like letters, digits, and symbols for making passwords.”

────────────────────────────────────────────────────────────────────────

IMPORT: csv
PURPOSE:
    Read and write CSV (Comma-Separated Values) files.

WHAT IT DOES IN THIS PROJECT:
    • Export password entries to a CSV backup file
    • Import entries from CSV into the database

DEFINITION:
    “Allows exporting and importing password data in spreadsheet format.”

────────────────────────────────────────────────────────────────────────

IMPORT: hashlib
PURPOSE:
    Provides PBKDF2-HMAC-SHA256 hashing functionality.

WHAT IT DOES IN THIS PROJECT:
    • Hashes the master password using PBKDF2 (120,000 iterations)
    • Compares hashes during login
    • Ensures master password is NEVER stored in plain text

WHY PBKDF2 IS USED:
    • Standard, secure, and part of Python’s standard library
    • Makes brute-force guessing extremely slow for attackers

DEFINITION:
    “Used to securely hash the master password so it cannot be read or reversed.”

────────────────────────────────────────────────────────────────────────

IMPORT: secrets
PURPOSE:
    Generates cryptographically secure random values.

WHAT IT DOES IN THIS PROJECT:
    • Creates a random salt for PBKDF2 hashing:
        secrets.token_bytes(16)
    • Ensures each user’s master password hash is unique
    • Protects against rainbow table attacks
    • Performs secure comparison of stored hash and derived hash

TEACHER-FRIENDLY DEFINITION:
    “Creates strong random values used in secure password hashing.”

────────────────────────────────────────────────────────────────────────

IMPORT: datetime, timezone
PURPOSE:
    Handle and generate timestamps.

WHAT IT DOES IN THIS PROJECT:
    • Adds date_added timestamps for entries
    • Records consistent ISO timestamps (UTC-safe)
    • Ensures editable logging for CSV exports/imports

DEFINITION:
    “Used to record when each entry was created.”

────────────────────────────────────────────────────────────────────────

IMPORT: tkinter as tk
PURPOSE:
    Python’s built-in GUI toolkit (no installation needed).

WHAT IT DOES IN THIS PROJECT:
    • Creates the main window
    • Manages buttons, labels, spacing, layout
    • Handles user input via variables (StringVar, IntVar)

DEFINITION:
    “Used to create the graphical user interface of the app.”

────────────────────────────────────────────────────────────────────────

IMPORT: ttk
PURPOSE:
    Provides modern “themed” GUI widgets.

WHAT IT DOES IN THIS PROJECT:
    • Makes the interface clean and professional
    • Implements:
        - Treeview (table listing passwords)
        - Buttons
        - Frames
        - Comboboxes
        - Checkbox toggles

DEFINITION:
    “Modern GUI elements for a cleaner interface.”

────────────────────────────────────────────────────────────────────────

IMPORT: messagebox
PURPOSE:
    Shows pop-up messages and warnings.

WHAT IT DOES IN THIS PROJECT:
    • Alerts for:
        - Success (“Saved”)
        - Errors (“Missing fields”)
        - Wrong master password
        - Confirmations (“Delete entry?”)

DEFINITION:
    “Used to show message pop-ups to the user.”

────────────────────────────────────────────────────────────────────────

IMPORT: simpledialog
PURPOSE:
    Prompt simple user input.

WHAT IT DOES IN THIS PROJECT:
    • Prompts:
        - Master password setup
        - Master password confirmation
        - Master password login

DEFINITION:
    “Used to ask the user to type the master password.”

────────────────────────────────────────────────────────────────────────

IMPORT: filedialog
PURPOSE:
    Open file browser windows.

WHAT IT DOES IN THIS PROJECT:
    • Lets user select a file for import (CSV)
    • Lets user choose where to save exports

DEFINITION:
    “Used to open files or save backups through a dialog box.”
